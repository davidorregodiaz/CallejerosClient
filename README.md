# CallejerosClient

CallejerosClient es la aplicación web desarrollada con **React** que interactúa con la **CallejerosAPI** para permitir a los usuarios:

- Visualizar animales disponibles para adopción.
- Postularse para adoptar y seguir el estado de sus solicitudes.
- Ver historias de adopciones y publicaciones especiales sobre animales.
- Recibir notificaciones y correos electrónicos sobre su actividad en la plataforma.

El frontend está enfocado en brindar una **experiencia interactiva, amigable y responsiva**, fomentando la adopción responsable y el cuidado de los animales.

## Instalación

1. Clonar el repositorio:

```bash
git clone git@github.com:davidorregodiaz/CallejerosClient.git
```

2. Entrar en la carpeta del proyecto:

```bash
cd tu-repo
```

3. Instalar las dependencias:

```bash
npm install
```

4. Ejecutar la aplicación

4.1. Para correr la aplicación en modo desarrollo:

```bash
npm start dev
```

5. Abre http://localhost:3000 en tu navegador.
